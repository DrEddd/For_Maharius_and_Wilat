Итератор на начало (begin), итератор на символ, следующий за последним (last), элемент(elem).
Возвращает итератор на первый элемент из диапазона от begin до last равный elem. Если такого нет, то возвращает last.
Да
vector<int>a;
int main () {
	for (int i=0; i<16; i++) a.push_back(15-i);;
	cout << find(a.begin(), a.end(), 5)-a.begin() << endl; //Выведет 10 - индекс элемента, равного 5.
	return 0;
};
