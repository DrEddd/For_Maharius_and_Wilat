Итератор на начало (begin), итератор на символ, следующий за последним (last), компоратор.
Возвращает итератор на первый элемент из диапазона от begin до last удовлетворяющий компаратору. Если такого нет, то возвращает last.
Да
bool comp (int a) {
	return a<5;
};
vector<int>a;
int main () {
	for (int i=0; i<16; i++) a.push_back(15-i);;
	cout << find(a.begin(), a.end(), comp)-a.begin() << endl; //Выведет 11 - индекс элемента, большего  5.
	return 0;
};
