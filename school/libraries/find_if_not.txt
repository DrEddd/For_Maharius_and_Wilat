Итератор на начало (begin), итератор на символ, следующий за последним (last), компоратор.
Возвращает итератор на первый элемент из диапазона от begin до last НЕ удовлетворяющий компаратору. Если такого нет, то возвращает last.
Нет
bool comp (int a) {
	return a>10;
};
vector<int>a;
int main () {
	for (int i=0; i<16; i++) a.push_back(15-i);;
	cout << find(a.begin(), a.end(), comp)-a.begin() << endl; //Выведет 5 - индекс элемента, не большего 10.
	return 0;
};
