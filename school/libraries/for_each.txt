Итератор на начало (begin), итератор на символ, следующий за последним (last), функция.
Выполняет функцию для каждого из элементов в диапазоне от begin до last.
Да
int a[] = {1,2,3,4,5};
void print(int b) {
	cout << b;
};
int main () {
	for_each(a, a+5, print); //Выведет 12345
	return 0;
};
